css로 무한 슬라이드 만드는 조건

  • slide 반복횟수를 2로 하면 .slider(제일 바깥)의 width가 .animation(애니메이션을 위한 요소)의 width / 2보다 작아야 위화감 없이 보임
  • slide 반복횟수를 3으로하면 .slider의 width가 .animation의 width * (2 / 3)보다 작아야 위화감 없이 보임
  • keyframes animation은 transform을 slide요소의 width * 갯수의 (-)만큼 움직여야함.
    만약 거꾸로 돌리고싶으면 reverse를 쓸 것.
