사업분야로그인플러스

서비스 정의

로그인 보안을 강화하여 부정 로그인을 차단, 개인정보를 안전하게 지키는 서비스. 웹과 모바일에서 모두 사용 가능한 넓은 커버리지.

서비스 주요기능
-불법 로그인 차단: 이용 사이트의 ID/PW 외에 로그인 플러스 서비스 가입 시 설정한 2차 암호의 사칙연산 값을 입력.
타인의 불법 로그인 차단 및 키보드 해킹 방지.
-로그인 실시간 알림: 등록 사이트 로그인 시, 실시간 SMS로 로그인 정보를 알림.

서비스 특징

기존의 ID, Password 만을 이용한 사이트로그인 방식보다 한층 강화된 "보안성"을 가진 '서비스암호 인증방식'과
'원채널(PC) 서비스인증'을 통해 '사이트로그인'이 이루어지는 "편의성"을 갖춘 서비스
[감수:고려대학교 정보보호대학원 이동훈 교수]
